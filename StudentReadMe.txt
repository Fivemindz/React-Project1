MyReads: React Book Tracking App

Description: This repository serves as a project submission for the MyReads: A Book Tacking App. 

Installation: 
  Step 1: Clone repository to local machine. Repository URL - https://github.com/Fivemindz/React-Project1.git
  Step 2: CD into starter folder.
  Step 3: Run npm install command.
  Step 4: Run npm start command.
  Application will run in a browser window @ http://localhost:3000
